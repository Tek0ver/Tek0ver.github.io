Obtenir les recettes d'un site

Obtenir les recettes d'un site

Récupérer les recettes de cuisine d'un site contenant un ingrédient spécifique et en stocker les informations dans un fichier.

Lien GitHub : https://github.com/Tek0ver/scraping-recipes

Scraping

Présentation

Le but est de scraper le site de recettes de cuisine suivant : https://recipes.lewagon.com/, puis de stocker le nom de chaque recette ainsi que la difficulté et le temps de préparation associé, mais uniquement les recettes contenant l’ingrédient voulu.

Fonctionnement

Tout ce passe avec un script Python (recipe.py) au travers de la console et en donnant l’ingrédient désiré, ici “Tea” pour l’exemple.

Le script demande de télécharger à l’aide de la commande curl la page en local si celle-ci n’est pas détecter, on peut ensuite y récupérer les informations en lancer le script.

Présentation

Le but de ce projet va être de récupérer tous les avis en français de plusieurs produits sur Amazon, la note, le titre, et le texte de chaque avis, pour ensuite faire de l’analyse de language et de l’analyse de sentiments, puis entraîner un modèle NLP pour trouver la note d’un avis avec seulement son texte.

Projet

Le projet se décompose en 4 étapes :

  • Le scraping : Récupérer les avis en Français automatiquement de plusieurs produits.
  • Exploration des données : Tirer des conclusions des données récupérées pour préparer l’étape suivante.
  • Machine Learning : Créer le modèle capable de prédire la note d’un avis en fonction de son texte.
  • Création d’une API : Mettre à disposition une API qui pourrait déduire la note à partir du texte d’un avis.
